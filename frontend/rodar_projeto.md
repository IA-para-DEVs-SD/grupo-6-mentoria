# Como rodar o projeto

Requer Node.js 20.19+ ou 22.12+.

## Primeira vez

```bash
# 1. Instalar dependências
npm install

# 2. Subir o servidor
npm run dev
```

## Próximas vezes

```bash
npm run dev
```

## Acessar

- App: http://localhost:5173

## Outros comandos

```bash
# Build de produção (com type-check)
npm run build

# Preview do build de produção
npm run preview
```
